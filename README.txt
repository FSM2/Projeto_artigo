Infraestrutura de dados do artigo: Levantamento Aerofotogramétrico e Construção de Modelos 3D dos 
Sítios de Registros Rupestres Gravados no Enclave Arqueológico Pedra Ferrada, Município de Jucurutu, 
Rio Grande do Norte

1 - Código python com parâmetros de processamento fotogramétrico de sítios de gravuras rupestres
baseado na biblioteca pyodm - Projeto OpenDroneMap: https://opendronemap.org
TOFFANIN, P. OpenDroneMap: The Missing Guide. MasseranoLabs LLC, 2019

2 - Código de visualização da nuvem de pontos, formato las, baseado nas bibliotecas laspy e pptk

3 - Disponibilização de alguns modelos tridimensionais gerados em formato obj

Para o processamentos fotogramétrico é preciso instalar o Docker, plataforma opensource que 
possibilita a criação e administração de ambientes no computador, a exemplo de uma máquina virtual, 
mas trabalhando com uma infraestrutura de container. Informações: https://www.docker.com
